# Pastel Themes

Pastel themes feature soft, muted colors that are easy on the eyes while maintaining visual distinction between elements.

## Available Themes

### Catppuccin Latte
- **Author**: Catppuccin
- **Colors**: Light pastel base with vibrant accents
- **Best For**: Light mode preference, daytime use
- **Pairs Well With**:
  - Catppuccin for VS Code (Latte variant)
  - Light desktop environments

### Rosé Pine
- **Author**: Rosé Pine
- **Colors**: Soft pinks and purples with warm tones
- **Best For**: Aesthetic-focused setups, creative work
- **Pairs Well With**:
  - Rosé Pine for Firefox
  - Rosé Pine for VS Code

## Usage Tips

- Pastel themes work well in bright environments
- Consider using with dark mode editor themes for contrast
- Great for presentations and screencasts
