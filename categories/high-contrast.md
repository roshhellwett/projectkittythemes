# High-Contrast Themes

High-contrast themes prioritize maximum visibility with sharp color distinctions and clear visual hierarchy.

## Available Themes

### Nord
- **Author**: arcticicestudio
- **Colors**: Cool arctic blues with polar night tones
- **Best For**: Maximum readability, cold color aesthetics
- **Pairs Well With**:
  - Nord for VS Code
  - Nord for Firefox
  - NordVPN aesthetic

### Solarized Dark
- **Author**: altercation (Ethan Schoonover)
- **Colors**: Classic low-blue light design
- **Best For**: Extended reading, reduced eye fatigue
- **Pairs Well With**:
  - Solarized for tmux
  - Solarized for Vim

## Usage Tips

- High-contrast themes are ideal for accessibility needs
- Better visibility in bright environments
- Recommended for users with visual impairments
- Solarized uses a unique philosophy of minimizing blue light
