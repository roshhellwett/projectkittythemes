# Dark Themes

Dark themes feature deep, rich backgrounds with vibrant accents designed for comfortable extended use in low-light environments.

## Available Themes

### Tokyo Night
- **Author**: enkia
- **Colors**: Blue-toned with purple accents
- **Best For**: Late-night coding sessions, Neovim users
- **Pairs Well With**: 
  - Neovim Tokyo Night plugin
  - VS Code Tokyo Night theme

### Catppuccin Mocha
- **Author**: Catppuccin
- **Colors**: Purple-toned with warm pink and green accents
- **Best For**: Catppuccin fans, unified desktop themes
- **Pairs Well With**:
  - Catppuccin for VS Code
  - Catppuccin for Firefox

### Gruvbox Dark
- **Author**: morhetz
- **Colors**: Warm browns and oranges with retro aesthetic
- **Best For**: Retro computing enthusiasts, warm color lovers
- **Pairs Well With**:
  - Vim Gruvbox plugin
  - i3status-gruvbox

## Usage Tips

- Dark themes reduce eye strain in dim environments
- Higher contrast versions available in High-Contrast category
- All dark themes support true color (24-bit)
